# 🌦️ Weather App – City Weather Finder

A responsive Weather Application built using HTML, CSS, and JavaScript that fetches real-time weather data using a Weather API.

Users can enter a city name to instantly view temperature, wind speed, and humidity displayed in clean UI cards.

---

## 🚀 Live Demo
https://majestic-alfajores-77b99d.netlify.app/

---

## 🛠 Tech Stack

- 🌐 HTML5
- 🎨 CSS3
- ⚡ JavaScript (ES6+)
- 🌍 Weather API (e.g., OpenWeatherMap API)

---

## ✨ Features

✔️ Search weather by city name  
✔️ Displays real-time data  
✔️ Temperature display  
✔️ Wind speed display  
✔️ Humidity display  
✔️ Clear previous search results  
✔️ Responsive card-based UI  
✔️ Dynamic DOM updates  
✔️ Error handling for invalid cities  

---

## 🧠 How It Works

1. User enters a city name.
2. JavaScript fetches weather data from the API.
3. The API returns:
   - Temperature
   - Wind speed
   - Humidity
4. Data is dynamically displayed inside cards below the search bar.
5. User can clear the results and search for another city.
